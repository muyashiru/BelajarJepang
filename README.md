# 🎌 Belajar Jepang - Website Latihan Gratis

Website interaktif untuk belajar Hiragana, Katakana, dan Kanji secara gratis tanpa perlu login!

## ✨ Fitur Utama

### 📝 Latihan Hiragana
- 46 karakter Hiragana lengkap
- Quiz interaktif dengan pilihan ganda
- Sistem penilaian otomatis
- Progress tracking

### 📝 Latihan Katakana  
- 46 karakter Katakana lengkap
- Format quiz yang sama dengan Hiragana
- Ideal untuk pemula

### 📚 Latihan Kanji
- **Kanji N5**: 103 karakter level pemula
- **Kanji N4**: 168 karakter level menengah
- **Mode Campuran**: Kombinasi N5 & N4
- Latihan mencakup:
  - Arti karakter
  - On-yomi (bacaan China)
  - Kun-yomi (bacaan Jepang)
  - Contoh penggunaan

### 📖 Halaman Referensi
- Tabel lengkap semua karakter
- Informasi detail untuk setiap Kanji
- Grid interaktif yang mudah dinavigasi

## 🎯 Total Konten Latihan

- **Hiragana**: 46 karakter
- **Katakana**: 46 karakter  
- **Kanji N5**: 103 karakter
- **Kanji N4**: 168 karakter
- **Total**: 363 karakter Jepang!

## 🚀 Cara Menggunakan

1. Buka `index.html` di browser
2. Pilih jalur belajar (Hiragana/Katakana/Kanji)
3. Mulai berlatih!
4. Tidak perlu install atau login

## 📊 Database Lengkap

### Hiragana (46 karakter)
Semua karakter dasar dari あ sampai ん, termasuk semua baris (a, ka, sa, ta, na, ha, ma, ya, ra, wa)

### Katakana (46 karakter)  
Semua karakter dasar dari ア sampai ン, untuk kata serapan asing

### Kanji N5 (103 karakter)
Mencakup:
- Angka (一、二、三、...)
- Hari & Waktu (日、月、火、...)
- Kata Dasar (人、本、国、...)
- Kata Kerja Umum (見、聞、食、...)
- Dan banyak lagi!

### Kanji N4 (168 karakter)
Level menengah dengan karakter yang lebih kompleks

## 🎮 Mode Quiz

1. **Quiz Standar**: 10-15 pertanyaan acak
2. **Pilihan Ganda**: 4 opsi jawaban
3. **Feedback Langsung**: Tanda centang/silang

## 💡 Tips Belajar

1. **Konsisten**: Latih 10-15 menit setiap hari
2. **Mulai dari Dasar**: Kuasai Hiragana dulu, lalu Katakana, baru Kanji
3. **Ulangi**: Review karakter yang sudah dipelajari
4. **Catat**: Tulis karakter di kertas untuk memorisasi lebih baik
5. **Gunakan Konteks**: Pelajari Kanji dengan contoh kata
